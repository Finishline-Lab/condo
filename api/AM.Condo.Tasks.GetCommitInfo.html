Commits

Gets or sets the commits that belong to the current release.

Declaration
public ITaskItem[] Commits { get; set; }
Property Value
Type Description
ITaskItem[]
Remarks

Will contain a list of all commits that comply with the indicated format. The SHA value will be the . In addition, the task items will contain metadata that describes any commit that follows the conventional-changelog style guide. This metadata includes the following:

  • Type The type (of change) for the commit, which may be Features, Performance Improvements, Bug Fixes, Documentation, Styles, Code Refactoring, Tests or Chores.
  • Scope The component in which the change occurred.
  • Subject The subject, or description, of the change that occurred.
  • Body The body that describes additional details about the change that occurred.
  • Closes A semi-colon (;) delimited list of issues (work items) that are closed (resolved) by the commit.
  • References A semi-colon (;) delimited list of references (work items) that are related to the commit.
  • Breaking A summary description of breaking changes associated with the commit.
  • Message The entire commit message including new-lines. This is useful for commits that do not follow the conventional changelog style.
